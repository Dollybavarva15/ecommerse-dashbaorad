# 📊 E-Commerce Sales Dashboard

## 📌 Project Overview
Interactive Power BI dashboard developed to analyze e-commerce sales performance, profit trends, customer behavior, and regional sales insights using dynamic visualizations and KPIs.

## ✨ Features
- 📈 Sales and Profit Analysis
- 📅 Monthly Sales Trends
- 🛍️ Category-wise Performance Analysis
- 👥 Customer Insights
- 🎛️ Interactive Filters and Visualizations
- 📊 KPI Tracking Dashboard

## 🛠️ Tools & Technologies Used
- ⚡ Power BI
- 📉 Data Visualization
- 📊 Data Analysis
- 📂 Excel / CSV Dataset

## 🔍 Key Insights
- 🏆 Identified top-performing product categories
- 📈 Analyzed monthly sales and profit growth
- 🌍 Compared regional sales performance
- 🛒 Tracked customer purchasing behavior

## 🎯 Purpose
The objective of this project is to transform raw e-commerce sales data into meaningful business insights through interactive dashboards and visual analytics.
